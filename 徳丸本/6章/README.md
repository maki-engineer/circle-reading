## 6.1 (525p) 文字コードとセキュリティの概要
### 文字コードとは
文字コードは次の2つの概念を合わせたもの。
- 文字集合 (6.2で説明)
- 文字エンコーディング (6.3で説明)

Webアプリケーション上で文字コードを指定する箇所はたくさんある。
- 入力の場面
  - ブラウザ → HTTPリクエスト
- 処理の場面
  - アプリケーションの処理
  - スクリプトファイル
- 出力の場面
  - HTTPレスポンス → HTMLブラウザ表示
  - DB (SQL)
  - ファイル
  - メール
  - 外部コマンド
